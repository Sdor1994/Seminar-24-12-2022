## Создания "коммита"
Для создания коммита используеться команда 'git commit". Для этого в терминале с папкой репозиторием необходимо написать "git commit -m <сообщение к коммиту>". Сообщение к коммиту писать ***ОБЯЗАТЕЛЬНО!!!***.
